WeatherPy: Weather Indicators by Latitude


Analysis of weather patterns is an extremely complex problem often requiring supercomputers for predictive capabilities. However, many basic analyses can be performed using simple correlative tools. As an example of this simplified approach, data was acquired and multiple measurements were evaluated including temperature, humidity, cloud cover, and wind speed as a correlative with the latitude of the city. Using this simple correlation, we demonstrate that correlated and non-correlated measurements are readily identified.





Summary of Study


This study used the OpenWeatherMap API and Python/Pandas data analysis tools to identify weather indicators for randomly generated cities. These weather indicators were plotted against the latitude of the city and graphically represented using the MatPlotLib graphical library. Additional analyses can be completed for randomly generated cities using the files in the Github repo linked below.
